# Instagram Reel Thumbnails

This folder contains thumbnail images for Instagram reels displayed in the hitchhiking map hover popups.

## Naming Convention
- `croatia-day1.jpg` - Day 1 hitchhiking from Croatia
- `austria-stop1.jpg` - First stop in Austria
- etc.

## Image Requirements
- **Aspect ratio**: 9:16 (vertical, like Instagram reels)
- **Size**: 400x720px recommended
- **Format**: JPG or PNG
- **Quality**: Optimized for web (under 200KB each)
